# Shisa-Kanko Updates

## Version 0.2.0 - UI & Experience Upgrade

### 1. Persistent HUD "Signal" Counter
- **Feature**: A persistent status bar item now displays the total number of active AI highlights across the workspace (e.g., `$(radio-tower) 3 Signals`).
- **Interaction**: Clicking the status bar item opens a Quick Pick menu listing all files with active highlights, allowing instant navigation.

### 2. Customizable HUD Style Profiles
- **Feature**: New configuration setting `shisa-kanko.hudStyle` to control the visual weight of the highlights.
- **Profiles**:
    - **High Visibility** (Default): Full line highlight with thick outline.
    - **Minimalist**: Thick gutter-only bars, transparent background.
    - **Underline**: Subtle wavy underline, minimal obstruction.
